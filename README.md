## Description

[Nest](https://github.com/nestjs/nest) framework TypeScript starter repository.

## Installation

```bash
$ npm install
```

## Install Nest Cli

```bash
$ npm i -g @nestjs/cli
```

## Up database

```bash
$ docker compose up -d
```

## Clone the **.env.template** file and rename to **.env** file

## Fill the variables at **.env** file

## Run seed

```bash
$ localhost:3000/api/v2/seed
```

## Running the app

```bash
# development
$ npm run start

# watch mode
$ npm run start:dev

# production mode
$ npm run start:prod
```

## Seed

```bash
localhost:3000/api/v2/seed
```

## Test

```bash
# unit tests
$ npm run test
```
